What is AGAUMb?

A Game About Ur Moms bananas is a series of RPG games made by JdavisBro.

How did you come up with the idea?

I didn't, I was in a group chat with F1Ethan and a random person that liked pigs very much. I asked them "What should I make a game about?" there replies where. F1Ethan: "ur mom" PigPerson420*: "Bananas". So I started work on the original game, 'A Game Aboout Ur Moms bananas'. At first, the about had two o's, but i dropped that and only took the non capital b in bananas.
